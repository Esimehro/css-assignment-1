Difference between Padding, Margin and Borders

Padding:

This is used to create space for the content within a container.

Margin:

This is used to create space outside of a container or between containers. Margins also specify space around an element's border.

Borders:

This allows you specify the width, style and color for an element. Border is the shorthand property for border-style, border-color, and borther width. It saves time because you can use a single css file on multiple html pages.
